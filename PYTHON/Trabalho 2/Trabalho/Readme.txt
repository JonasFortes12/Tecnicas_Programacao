A questão 1 consiste em dois arquivos: mainNComplexo.py e modelagemNComplexo.py

	O programa deve ser executado no arquivo mainNComplexo.py
	Os testes estão divididos por comentários #.
	
	Para fazer testes com valores diferentes, basta alterar os parâmetros
	dos métodos de inicialização do número.
	
	O número imaginário também pode ser inicializado ao instanciar o objeto.
	n1 = ModelagemNComplexo(1) # parte real somente
	n1 = ModelagemNComplexo(1, 2) # parte real e imaginária

A questão 2 consiste no arquivo criptografia.py
	
	Ao execultar o programa, deve-se digitar apenas valores inteiros de 4 dígitos
	e escolher uma opção: [1]-Encriptografar  [2]-Descriptografar
	Caso contrário, o programa informará uma mensagem de erro "Opcao Invalida ou Numero Invalido ."
